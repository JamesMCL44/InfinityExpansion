# 無限附加
此附加添加了非常多的終局遊戲至黏液科技, 增加了 '無限錠' 這需要消耗大量的資源與精力來製作, 並用於製作功能強大的機器與裝備.

此為**非官方**版本, 請勿在該作者的問題追蹤內回報!
[原作者連結](https://github.com/Mooy1/InfinityExpansion)
